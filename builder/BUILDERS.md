## 📋 Overview

This catalog lists all available HTML prompt builders in the collection. Each builder is a standalone HTML file that generates randomized prompts for AI image generation, specifically optimized for SDXL and ComfyUI workflows.

### Quick Reference
- **Ultra Series** (7) - Advanced dual-toggle builders with extensive customization
- **Legacy Series** (3) - Original simpler builders, still functional
- **Specialized** (3) - Unique purpose-built builders

## 🚀 How to Use These Builders

### Opening a Builder
1. Navigate to `builder/` directory
2. Double-click any `.html` file
3. Opens in your default web browser
4. No installation or server required!

### Using a Builder
1. **Select options** from dropdown menus (or leave on "Random")
2. Click **"Generate Prompt"** button
3. **Copy** the generated prompt to clipboard
4. **Paste** into ComfyUI or your AI image generator
5. **Regenerate** for different variations

### Tips for Best Results
- Mix "Random" and specific selections for variety
- Use toggles to switch between styles/ratings
- Generate multiple times to compare options
- Copy interesting combinations for later use
- Customize the HTML if you want different options

## 🔧 Customizing Builders

All builders are self-contained HTML files with embedded JavaScript. To customize:

1. **Open in text editor** (VS Code, Notepad++, etc.)
2. **Find the category arrays** (usually near the top of `<script>` section)
3. **Add/remove/modify** options in the arrays
4. **Save** the file
5. **Refresh** in browser to see changes

**Example:**
```javascript
const outfits = [
    "leather jacket",
    "your custom outfit here",
    "another option"
];
```
